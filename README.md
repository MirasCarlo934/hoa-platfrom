# hoa-platfrom
A platform for HOA

## Project Setup

1. **Install dependencies**
   ```sh
   npm install
   ```

2. **Compile TypeScript**
   ```sh
   npx tsc
   ```

3. **Run the server**
   ```sh
   node dist/app.js
   ```
   Or, for development with auto-compile:
   ```sh
   npm run dev
   ```

4. **Run tests**
   ```sh
   npm test
   ```

## Deployment

The app will be available at http://localhost:3000
